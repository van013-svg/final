Flood Atlas: Mapping Flood-Related Climate Conditions

This interactive visualization uses regional historical precipitation data to explore annual trends, precipitation variances, and climate variables near major flood basins.

Project Overview

This project uses regional meteorological data to examine climate conditions that contribute to catastrophic basin overflows. Instead of focusing on simple average trends, the dashboard exposes multi-metric profiles across target impact areas.

Use the event selectors below to highlight historical zones, scrub the time framework slider across months, and swap visualization variables to watch regional climate dynamics react.
